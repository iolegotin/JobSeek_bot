## Что сделать: 
1. В `app/core/services/hh_parser.py` создать класс `HHAsyncClient`.
2. Реализовать метод `fetch_vacancies(keyword)`, делающий асинхронные HTTP-запросы через `httpx` или `aiohttp`.
3. Добавить базовую обработку статус-кодов и парсинг сырого JSON ответа API.
## Ожидаемый результат: Клиент, способный делать запросы к API HH.ru и возвращать сырые данные вакансий.
## Как протестировать: Скрипт `test_hh_client.py`: делает 1 асинхронный request с валидным ключом, проверяет статус-код `200` и наличие поля `items` в ответе.
