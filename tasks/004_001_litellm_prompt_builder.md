# TASK 004_001_litellm_prompt_builder
## Назначение: Интеграция `LiteLLM` и динамическая сборка системного промпта.
## Входные данные: Настроенный `.env` с ключем LiteLLM/Local LLM.
## Что сделать: 
1. В `app/core/services/litellm_svc.py` создать обёртку вызова модели.
2. Реализовать сборку динамического промпта из `query_text` и весов пользователя (`weights_config`).
3. Строгая валидация JSON-ответа модели (score 0.0-1.0, reasons, updated_weights_suggestion).
## Ожидаемый результат: Сервис, отдающий структурированный JSON от LLM по вакансии и профилю юзера.
## Как протестировать: Скрипт делает dry-run вызова модели с тестовым профилем и текстом вакансии, парсит JSON-ответ, проверяет наличие ключа `score` в диапазоне `[0.0; 1.0]`.
